# Rajshree Jewellers

[**View Live Website**](https://teqkilasolutions.github.io/rajshree/)

A luxury jewelry website for Rajshree Jewellers, featuring the exclusive "Kada King" collection.

## Features

-   **Live Gold Rates:** Fetches real-time gold and silver rates via API.
-   **Dynamic Pricing:** Product prices are calculated automatically based on live rates, weight, making charges (15%), and GST (3%).
-   **Product Catalog:** Browse collections including Kadas, Necklaces, Rings, and more.
-   **Shopping Cart & Wishlist:** Functional cart and wishlist with local storage persistence.
-   **User Authentication:** Basic login/signup simulation.
-   **Responsive Design:** Fully responsive layout for mobile and desktop.
-   **Luxury UI:** Custom cursor, magnetic buttons, and smooth scroll animations.

## Technologies Used

-   HTML5
-   CSS3 (Custom Properties, Flexbox, Grid)
-   JavaScript (ES6+, Fetch API, LocalStorage)
-   Font Awesome (Icons)
-   Google Fonts (Cormorant Garamond, Montserrat)

## Setup

1.  Clone the repository.
